# Transparency Dashboard

Transparency is the trust layer of BLOCKWORLD:  
players verify fairness, sponsors verify delivery, investors verify diligence.

## What you can see

Typical modules (as shown on the website/app):

- Burned Today
- Treasure Pool
- Referral Paid Today
- Treasury

Plus:

- Live Token Flow (User Spend -> Pool/Burn/Referral/Treasury)
- Recent Transactions (Tx Feed)

## v1.0 split recap

- Revive (10 BWT): 50% Pool / 30% Burn / 10% Referral / 10% Treasury  
- Endless Ticket (20 BWT): 50% Pool / 30% Burn / 10% Referral / 10% Treasury  
- Item Packs: 50% Burn / 10% Referral / 40% Treasury  

If anything differs on the site, the latest in-app rules prevail.
